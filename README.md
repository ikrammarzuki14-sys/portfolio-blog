## Overview

This project is a personal portfolio website for Ikram Marzuki, a Software Development student. The site presents selected projects, learning reflections, contact details, and an in-progress Final Year Project.

The portfolio has been redesigned to focus on web development, backend systems, databases, and AI-assisted career guidance.

An incomplete Final Year Project focused on a web-based career guidance platform for SPM leaver students. The project explores artificial intelligence, machine learning concepts, the RIASEC model, and decision tree logic for career prediction and education pathway recommendations.
